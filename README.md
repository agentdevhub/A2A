## 开启Agent间协作新时代：全新开放协议解锁智能体协同场景

<img src="images/a2a_main.png" width="70%" style="margin:20px auto;display:block;">

* **无缝Agent协作**：针对不同框架和厂商构建的自主、不透明Agent，推出标准化通信协议，有效解决现有Agent互操作性问题，实现跨平台智能体间以及与用户的高效协作。
* **简化企业级Agent集成**：提供智能Agent与企业现有应用的直连方案，支持企业将Agent能力无缝融入技术生态。
* **满足企业核心需求**：提供企业级安全协作所需的核心功能，包括能力发现、用户体验协商、任务与状态管理等关键模块。

## 连接智能体的开放标准

<img src="images/a2a_mcp_readme.png" width="70%" style="margin:20px auto;display:block;">

* **MCP（模型上下文协议）** - 工具与资源连接标准
  * 通过结构化输入/输出实现Agent与工具、API及资源的连接
  * Google ADK 原生支持MCP工具，兼容各类MCP服务端
* **A2A（Agent间通信协议）** - 智能体协作标准
  * 支持异构Agent间动态、多模态通信（内存、资源和工具保持隔离）
  * 社区驱动的开放标准
  * 提供基于Google ADK、LangGraph、Crew.AI的参考实现

## 反馈与演进

A2A协议目前处于开发阶段，我们将根据社区反馈持续优化。本仓库包含初始规范、技术文档及[示例代码](https://github.com/google/A2A/tree/main/samples)，后续将陆续增加新功能、典型案例、技术规范和开发库。待协议规范与示例代码达到生产级质量标准后，我们将发布1.0稳定版本。
